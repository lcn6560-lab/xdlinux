**Instruções de Instalação**

1.  **Baixe o pacote:**
    wget https://github.com/lcn6560-lab/xdlinux/blob/main/xdpkg.tar.gz

2.  **Descompacte o arquivo:**
    tar -vzxf xdpkg.tar.gz

3.  **Entre nos diretórios:**
    cd xdpkg
    cd ubiquity-slideshow-lubuntu
    cd usr
    cd share
    cd doc

4.  **Copie o arquivo para o diretório de destino:**
    cp -r ubiquity-slideshow.tar.gz /usr/share/doc

5.  **Vá até o diretório /usr/share/doc:**
    cd /usr/share/doc

6.  **Descompacte o arquivo:**
    tar -vzxf ubiquity-slideshow.tar.gz

7.  **Remova o arquivo compactado:**
    rm ubiquity-slideshow.tar.gz

8.  **Volte e copie o arquivo novamente:**
    cd /home/seu_usuario/xdpkg/ubiquity-slideshow-lubuntu/usr/share
    cp -r ubiquity-slideshow.tar.gz /usr/share

9.  **Vá até o diretório /usr/share:**
    cd /usr/share

10. **Descompacte o arquivo:**
    tar -vzxf ubiquity-slideshow.tar.gz

11. **Remova o arquivo compactado:**
    rm ubiquity-slideshow.tar.gz
